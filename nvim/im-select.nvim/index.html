im-select.nvim Neovim im-select packer.nvim ​

轉換模式時自動切換輸入法。

相依 ​

  • 僅可安裝於 Neovim 。
  • 需安裝 im-select 。
  • 使用 packer.nvim 管理。

安裝 ​

lua
use({
  'keaising/im-select.nvim',
  config = function()
    require('plugins.configs.im-select')
  end,
})

配置 ​

lua
im_select.setup()

慣用方式 ​

在寫文章時，會在進入 insert 模式時，將輸入法設為中文。這樣一來在回到 normal 模式時會自動切換回英文輸入法。
